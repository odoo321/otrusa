Odoo Inventory Adjustment Cancel/Reverse

Need to Correct inventory adjustment and Opening stock entries?. Its obvious that user can make mistake when entering data. If you made mistake on entering stock on Odoo by inventory adjustment ? Don't worry, Odoo Inventory Adjustment Cancel and Reverse app helps to reverse inventory adjustments in any stage. With using of Cancel Stock Inventory adjustment app, it allow to cancel inventory on done stage and reset inventory in draft stage. Cancel Button on Inventory adjustment only allow for specfic user group i.e Stock inventory adjustment cancel. Access of this group of user able to cancel inventory adjustment on done stage and set it to draft stage after that user can easily ractify thier mistake and place the correct stock.

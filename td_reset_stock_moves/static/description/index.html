Delete / Cancel / Reset Wrong Moves.

Can delete/cancel "Done" moves.

Auto Balance Inventory.

How To Activate This Feature ?

Navigate to Users > Activate the group 'Delete/Cancel Wrong Stock Moves'.

Select Multiple Stock Moves To Delete/Cancel.

Navigate to Inventory > Inventory > Reports > Stock Moves.

Choose Options to Delete | Cancel Moves.

Deleted Stock Moves | Auto Adjust stock Quantity.
